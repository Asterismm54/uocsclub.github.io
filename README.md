# University of Ottawa Computer Science Club

The uOCSClub github page.

## Running Locally

Our website is built using Jekyll

```bash
cd uocsclub.github.io
bundle install
bundle exec jekyll serve
```
Then check out your [local server](http://localhost:4000).
